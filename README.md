# Currency Converter :

A simple and user-friendly Currency Converter application built with JavaScript that allows users to convert one currency into another using real-time exchange rates.

# Features :
Convert between multiple currencies
Real-time exchange rate data (via API)
Clean and responsive UI
Easy to use for beginners
Error handling for invalid inputs
Show country flag with currency

# Technologies Used :
HTML
CSS
JavaScript (Vanilla JS)
Exchange Rate API

# How It Works :
Select the base currency 
Select the target currency 
Enter the amount
Click the "Convert" button
The converted amount will be displayed instantly

# API Used :
This project uses a free currency exchange API to fetch real-time data.
